# Security Policy

## Scope

This repository contains documentation and templates only — no executable code. Security concerns are limited to:

- Sensitive data accidentally committed (credentials, internal URLs, PII)
- Malicious content injected via pull requests

## Reporting

If you discover sensitive data or a security concern, please report it privately:

1. **Do not** open a public issue
2. Use [GitHub Security Advisories](https://github.com/dpetrakov/architect-handbook/security/advisories/new) to report privately
3. Or email: dpetrakov@hotmail.com

We will respond within 7 days.
