# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2025-03-13

### Added

- Core Standard: rigor levels L0-L3, artifact matrix, Definition of Done
- 5 architect role profiles with competency matrices and RACI
- 8-phase architectural process with AI augmentation
- 30+ artifact catalog
- 18 practices: ADR, review, NFR, fitness functions, security, FinOps, platform engineering, AI governance, and more
- 7 engagement playbooks: audit, discovery, delivery, platform, modernization, cloud migration, transformation
- Governance charter: decision rights, ARB, escalation, waivers
- Quality gates: CI/CD gates, review checklists, phase gates
- Tooling profiles by rigor level
- Implementation roadmap with 3 horizons
- 33 artifact templates
- State-of-the-art research 2024-2026
